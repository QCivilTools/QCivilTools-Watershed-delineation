about=QCT Watershed Delineation is part of the QCivilTools civil engineering plugin suite.
    Two-phase automated watershed delineation powered by WhiteboxTools (WBT):

    PHASE 1 (Steps 1-5): Fill DEM → D8 Flow Direction → Flow Accumulation → Extract Streams → Vectorize Stream Network.
    PHASE 2 (Steps 6-13): Snap Pour Points → Delineate Watershed → Vectorize → UnnestBasins → Longest Flow Path → Subbasins → Subbasin Info shapefile → All-DEM Subbasins shapefile.

    Outlet points can be picked interactively on the QGIS map canvas.
    All output layers share the CRS of the input DEM automatically.
